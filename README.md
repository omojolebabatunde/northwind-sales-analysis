# Northwind Sales Analysis Dashboard

## 📊 Project Overview
This project analyzes the Northwind dataset to uncover key business insights across sales, customers, and product performance. The dashboard was built using Power BI to support data-driven decision-making.

## 🚀 Objectives
- Analyze sales performance over time
- Identify top-performing products and categories
- Understand customer purchasing behavior
- Evaluate regional sales distribution

## 🛠 Tools Used
- Power BI (Data Visualization)
- DAX (Measures & KPIs)
- Excel (Data Cleaning)

## 📈 Key Insights
- Top-performing product category contributed the highest share of total revenue
- Sales showed seasonal trends with peak periods
- A small group of customers generated a large percentage of revenue (Pareto principle)

## 📊 Dashboard Features
- Revenue, Profit, and Orders KPIs
- Sales trend over time
- Top products and categories
- Regional performance analysis
- Customer segmentation
